# Nest-Guide-Restapi

## Config Environment

```bash
cp .env.example .env
```

## Installation

```bash
yarn
```

## Mock DB

```bash
docker network create HEEPOKE
```

```bash
docker compose up -d
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev
```

## Format Code

```bash
yarn format
```
